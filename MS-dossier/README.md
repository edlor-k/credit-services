# Dossier Microservice (MS-dossier)

## Описание

**MS-dossier** — микросервис для отправки сообщений по электронной почте.
Получает необходимую информацию от MS-deal, формирует и отправляет письма пользователям.

---

---

## Стек технологий

- **Java 21**
- **Spring Boot 3**
- **Lombok**
- **SLF4J** — логирование
- **Swagger** (springdoc-openapi) — документация API
- **Docker** — контейнеризация
- **Kafka** — для асинхронной отправки сообщений

---

## Структура проекта

- **/dto** — DTO-запросы и ответы
- **/model/entity** — сущности
- **/model/enums** — перечисления (статусы, роли и др.)
- **/mapper** — MapStruct-мапперы
- **/service** — интерфейсы бизнес-логики
- **/service/impl** — реализации бизнес-логики
- **/util** — утилиты и константы
- **/config** — конфигурация приложения

---

## Взаимодействие с внешними сервисами

- Интеграция с микросервисом Сделка (`MS-deal`) для получения данных, необходимых для формирования писем.
- Взаимодействие с микросервисом Deal осуществляется через Kafka, что позволяет асинхронно обрабатывать запросы и отправлять уведомления пользователям.
