# Кредитный калькулятор
## Описание проекта

**Кредитный калькулятор** — backend-приложение, реализующее прототип кредитного банка на основе микросервисной архитектуры.

Система автоматизирует обработку кредитных заявок, расчёт кредитных условий и уведомление пользователя об этапах, обеспечивая полный цикл одобрения и выдачи кредита.

## Цель проекта

Автоматизация процесса оформления кредита: от подачи заявки и скоринга до выдачи кредита и формирования документов.  
Система обеспечивает прозрачность, отказоустойчивость и масштабируемость обработки кредитных заявок.

## Архитектура и стек

Приложение состоит из следующих микросервисов:

- **Gateway** — точка входа, маршрутизация клиентских запросов.
- **Deal** — обработка кредитных заявок и управление процессами.
- **Statement** — приём и проверка кредитных заявок.
- **Dossier** — хранение документов и взаимодействие с email (SMTP).
- **Calculator** — расчёт кредитных условий.
- **Database** — централизованное хранение данных.

**Взаимодействие сервисов:**  
REST API и асинхронный обмен через **Kafka**.

**Используемый стек:**  
- Spring Boot (REST API)
- PostgreSQL (база данных)
- Kafka (брокер сообщений)
- Swagger/OpenAPI (документирование API)
- Docker (контейнеризация)
- CI/CD (автоматизация сборки и деплоя)
