# Кредитный конвейер

Кредитный конвейер — проектное задание Neoflex, направление Java-разработка.

## Цель проекта

Получить практический опыт работы со Spring, PostgreSQL, Kafka и построением микросервисной архитектуры.

## Описание

В рамках проекта реализуется backend-приложение, состоящее из набора микросервисов:

- Gateway — точка входа для клиента, маршрутизация запросов
- Deal — основной сервис для обработки заявок и управления процессами
- Statement — сервис подачи и проверки кредитной заявки
- Dossier — хранение документов, взаимодействие с email (SMTP)
- Calculator — расчет кредитных условий
- Database (db) — централизованное хранение данных

Между сервисами используются REST API и асинхронное взаимодействие через Kafka.

## Приобретаемые навыки

- REST-API на Spring Boot
- БД и инструменты миграции
- Микросервисная архитектура (разделение ответственности, взаимодействие сервисов)
- Синхронное и асинхронное взаимодействие сервисов (Kafka)
- Документирование API через Swagger/OpenAPI
- Основы CI/CD и контейнеризации
